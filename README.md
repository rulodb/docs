# RuloDB documentation

Documentation for RuloDB.

## Getting Started

1. Install dependencies:
    ```shell
    npm install
    ```
2. Start the development server:
    ```shell
    npm run dev
    ```
3. Open your browser at http://localhost:3000 to view the documentation site.
