To check if two values are not equal, use the `ne` function, which returns `true` if the two values are not equal and `false` otherwise.

## Syntax

<CodeGroup>

```ts TypeScript
value.ne(otherValue) â†’ result
```

</CodeGroup>

### Arguments

List of arguments to provide.

<ParamField body="otherValue" type="string | number" required>
  The value to compare with. The type of the value being compared depends on the value used for
  comparison.
</ParamField>

### Optargs

No optional arguments.

### Returns

<ParamField body="result" type="boolean">
  Returns `true` if the two values are not equal and `false` otherwise.
</ParamField>

## Behavior

No special behavior.

## Notes & Caveats

No notes.

## Example

### Return users whose firstname is not "John"

Filter for users in the `users` table where the user's `first_name` is not `John`.

<CodeGroup>

```ts TypeScript
await r
  .db()
  .table('users')
  .filter((doc) => doc.first_name.ne('John'))
  .run(client);
```

</CodeGroup>

### Filter orders outside of Europe

Filter for orders in the `orders` table where the order's shipping address `continent` is not `Europe`, where the continent is a nested field of `address`.

<CodeGroup>

```ts TypeScript
await r
  .db()
  .table('orders')
  .filter((doc) => doc.field('address.continent').ne('Europe'))
  .run(client);
```

</CodeGroup>
