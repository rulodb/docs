To check if two values are less than or equal to each other, use the `le` function, which returns `true` if the first value is less than or equal to the second value and `false` otherwise.

## Syntax

<CodeGroup>

```ts TypeScript
value.le(otherValue) â†’ result
```

</CodeGroup>

### Arguments

List of arguments to provide.

<ParamField body="otherValue" type="string | number" required>
  The value to compare with. The type of the value being compared depends on the value used for
  comparison.
</ParamField>

### Optargs

No optional arguments.

### Returns

<ParamField body="result" type="boolean">
  Returns `true` if the first value is less than or equal to the second value and `false` otherwise.
</ParamField>

## Behavior

No special behavior.

## Notes & Caveats

No notes.

## Example

### Filter low value orders

Filter for orders in the `orders` table where the order's `value` is less than or equal to `100`.

<CodeGroup>

```ts TypeScript
await r
  .db()
  .table('orders')
  .filter((doc) => doc.value.le(100))
  .run(client);
```

</CodeGroup>
