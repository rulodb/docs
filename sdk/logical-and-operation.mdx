To compare two values for logical AND operation, use the `and` function, which returns `true` if both values are `true` and `false` otherwise.

## Syntax

<CodeGroup>

```ts TypeScript
value.and(otherValue) â†’ result
```

</CodeGroup>

### Arguments

List of arguments to provide.

<ParamField body="otherValue" type="boolean" required>
  The value to compare with.
</ParamField>

### Optargs

No optional arguments.

### Returns

<ParamField body="result" type="boolean">
  Returns `true` if both values are `true` and `false` otherwise.
</ParamField>

## Behavior

No special behavior.

## Notes & Caveats

No notes.

## Example

### Filter adulst in the USA

Filter for users in the `users` table where the user's `age` is greater than or equal to `21` and the user's `country` is `USA`.

<CodeGroup>

```ts TypeScript
await r
  .db()
  .table('users')
  .filter((doc) => doc.age.ge(21).and(doc.country.eq('USA')))
  .run(client);
```

</CodeGroup>
