To check if two values are greater than each other, use the `gt` function, which returns `true` if the first value is greater than the second value and `false` otherwise.

## Syntax

<CodeGroup>

```ts TypeScript
value.gt(otherValue) â†’ result
```

</CodeGroup>

### Arguments

List of arguments to provide.

<ParamField body="otherValue" type="string | number" required>
  The value to compare with. The type of the value being compared depends on the value used for
  comparison.
</ParamField>

### Optargs

No optional arguments.

### Returns

<ParamField body="result" type="boolean">
  Returns `true` if the first value is greater than the second value and `false` otherwise.
</ParamField>

## Behavior

No special behavior.

## Notes & Caveats

No notes.

## Example

### Filter high value orders

Filter for orders in the `orders` table where the order's `value` is above `1 000 000`.

<CodeGroup>

```ts TypeScript
await r
  .db()
  .table('orders')
  .filter((doc) => doc.value.gt(1_000_000))
  .run(client);
```

</CodeGroup>
