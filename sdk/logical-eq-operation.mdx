To check if two values are equal, use the `eq` function, which returns `true` if the two values are equal and `false` otherwise.

## Syntax

<CodeGroup>

```ts TypeScript
value.eq(otherValue) â†’ result
```

</CodeGroup>

### Arguments

List of arguments to provide.

<ParamField body="otherValue" type="string | number" required>
  The value to compare with. The type of the value being compared depends on the value used for
  comparison.
</ParamField>

### Optargs

No optional arguments.

### Returns

<ParamField body="result" type="boolean">
  Returns `true` if the two values are equal and `false` otherwise.
</ParamField>

## Behavior

No special behavior.

## Notes & Caveats

No notes.

## Example

### Filter orders from the UAE

Filter for orders in the `orders` table where the order's shipping address `country` is `UAE`, where the country is a nested field of `address`.

<CodeGroup>

```ts TypeScript
await r
  .db()
  .table('orders')
  .filter((doc) => doc.field('address.country').eq('UAE'))
  .run(client);
```

</CodeGroup>
