To check if two values are greater than or equal to each other, use the `ge` function, which returns `true` if the first value is greater than or equal to the second value and `false` otherwise.

## Syntax

<CodeGroup>

```ts TypeScript
value.ge(otherValue) â†’ result
```

</CodeGroup>

### Arguments

List of arguments to provide.

<ParamField body="otherValue" type="string | number" required>
  The value to compare with. The type of the value being compared depends on the value used for
  comparison.
</ParamField>

### Optargs

No optional arguments.

### Returns

<ParamField body="result" type="boolean">
  Returns `true` if the first value is greater than or equal to the second value and `false`
  otherwise.
</ParamField>

## Behavior

No special behavior.

## Notes & Caveats

No notes.

## Example

### Filter users that are older than 21

Filter for users in the `users` table where the user's `age` is greater than or equal to `21`.

<CodeGroup>

```ts TypeScript
await r
  .db()
  .table('users')
  .filter((doc) => doc.age.ge(21))
  .run(client);
```

</CodeGroup>
