To check if two values are less than each other, use the `lt` function, which returns `true` if the first value is less than the second value and `false` otherwise.

## Syntax

<CodeGroup>

```ts TypeScript
value.lt(otherValue) â†’ result
```

</CodeGroup>

### Arguments

List of arguments to provide.

<ParamField body="otherValue" type="string | number" required>
  The value to compare with. The type of the value being compared depends on the value used for
  comparison.
</ParamField>

### Optargs

No optional arguments.

### Returns

<ParamField body="result" type="boolean">
  Returns `true` if the first value is less than the second value and `false` otherwise.
</ParamField>

## Behavior

No special behavior.

## Notes & Caveats

No notes.

## Example

### Filter users that are younger than 21

Filter for users in the `users` table where the user's `age` is under `21`.

<CodeGroup>

```ts TypeScript
await r
  .db()
  .table('users')
  .filter((doc) => doc.age.lt(21))
  .run(client);
```

</CodeGroup>

### Filter low value orders

Filter for orders in the `orders` table where the order's `value` is less than `100`.

<CodeGroup>

```ts TypeScript
await r
  .db()
  .table('orders')
  .filter((doc) => doc.value.lt(100))
  .run(client);
```

</CodeGroup>
