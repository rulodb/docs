<Danger>**Data Loss Warning**: This operation permanently deletes all data in the database.</Danger>

## Syntax

<CodeGroup>

```ts TypeScript

```

</CodeGroup>

### Arguments

List of arguments to provide.

<ParamField body="name" type="string" required>
  Name of the database to drop. See [notes](#notes-%26-caveats) for restrictions.
</ParamField>

### Optargs

No optional arguments.

### Returns

<ResponseField name="result" type="object">
  Information about the operation.

  <Expandable title="properties">
    <ResponseField name="dropped" type="number">
        The number of databases dropped (should be 1 for successful operations).
    </ResponseField>
  </Expandable>
</ResponseField>

## Behavior

- When the database is dropped, all tables associated with the database are also dropped.

## Notes & Caveats

- The database name must **not** be a reserved database name. The following names are reserved: `__system__`.

## Example

### Dropping a Database

Drop a database called `test`.

<CodeGroup>

```ts TypeScript
await r.dbDrop('test').run(client);
```

</CodeGroup>
