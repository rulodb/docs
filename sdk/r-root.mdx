In Rulo, the `r` object is the root object to build queries. It provides a set of functions to perform various operations on data. Every query starts with the `r` object.

## Syntax

<CodeGroup>

```ts TypeScript

```

</CodeGroup>

### Arguments

No arguments.

### Optargs

No optional arguments.

### Returns

<ParamField body="builder" type="object">
  The query builder object.
</ParamField>

## Behavior

No special behavior.

## Notes & Caveats

No notes.

## Example

### Using the query builder

Filter for `orders` where the order's `status` is `shipped` or `delivered`.

<CodeGroup>

```ts TypeScript
import { Client, r } from '@rulodb/rulodb';

const client = new Client();
await client.connect();

// Use `r` to build a query and execute it
await r.dbCreate('myDb').run(client);

await client.disconnect();
```

</CodeGroup>
