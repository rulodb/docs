To compare two values for logical OR operation, use the `or` function, which returns `true` if either value is `true` and `false` otherwise.

## Syntax

<CodeGroup>

```ts TypeScript
value.or(otherValue) â†’ result
```

</CodeGroup>

### Arguments

List of arguments to provide.

<ParamField body="otherValue" type="boolean" required>
  The value to compare with.
</ParamField>

### Optargs

No optional arguments.

### Returns

<ParamField body="result" type="boolean">
  Returns `true` if either value is `true` and `false` otherwise.
</ParamField>

## Behavior

No special behavior.

## Notes & Caveats

No notes.

## Example

### Filter for shipped or delivered orders

Filter for `orders` where the order's `status` is `shipped` or `delivered`.

<CodeGroup>

```ts TypeScript
await r
  .db()
  .table('orders')
  .filter((doc) => doc.status.eq('shipped').or(doc.status.eq('delivered')))
  .run(client);
```

</CodeGroup>
